### Exercises 2.02

```c

int main(void)
{
    printf("Parkison's Law:\nWork expands so as to");
    printf("fill the time \n");
    printf("available for its completion.\n");
    return 0;
}
```

(a) identify the directives and statements in this prgram.
(b) What output does the program produce?

